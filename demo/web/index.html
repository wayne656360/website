Off Menu Taiwan 星級主廚的私房菜單

10顆米其林星★13家矚目餐廳★展現對台灣的熱愛

2019年首次在香港成功舉辦的Off Menu美食嘉年華，受到美食愛好者的追捧與好評，今年的12月4日至6日將在台北舉辦，也是2020年唯一在亞洲舉辦的一場Off Menu。
美食是種態度、也是我們重要的生活文化特色；Tatler Taiwan此次以「Heritage」台灣在地文化為主題，邀請到13家享譽盛名的餐廳，包括了7位星級主廚、3位新銳潛力主廚以及榮獲亞洲五十大酒吧的調酒師加上一位難求的私廚甜點主廚，以台灣在地食材，為你獻上絕無僅有的13道在菜單上不會出現、令人驚豔的創意跨界料理，以台灣好風味向這片土地致敬！ 為能在這塊土地好好生活慶祝，
12/4~12/6 我們 OFF MENU TAIWAN 見！

Gaggenau

MARTELL

全球知名法國干邑白蘭地品牌馬爹利，於1715 年由貴族尚‧馬爹利所創立，為歷史最悠久的頂級干邑酒廠。三個世紀以來，馬爹利獨到的釀酒藝術創造了高品質與獨特的口感並享譽全球。馬爹利家族從葡萄栽培、釀酒程序、蒸餾過程之控制、和窖藏陳年之要求，每一步驟都堅持嚴格的品質管制，以及運用獨步全球的特殊秘方，創造出馬爹利干邑口感香醇，品味優雅的特質，確保市場地位屹立不搖，歷久彌堅。

S.Pellegrino天然氣泡礦泉水

擁有120年銷售歷史的聖沛黎洛天然氣泡礦泉水，源自義大利北方阿爾卑斯山麓，透過地底2000英呎深的石灰岩與火山岩層層過濾，歷經30年，最終於聖沛黎洛小鎮湧出，口感清爽、氣泡細緻，富含合諧與均衡穩定的天然礦物鹽，包括硫酸鹽、氫氣、碳酸鹽、氧化矽及鎂等，以細膩的氣泡和微妙的礦泉芬香獨樹一幟，有「礦泉水中的香檳」之美譽，為全球80%米其林、世界最佳50餐廳選用的「最佳的佐餐水」。
